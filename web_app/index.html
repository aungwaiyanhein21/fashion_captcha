Can Bot beat Human in CAPTCHA?

Introduction


One of the ways to prevent bots scraping the website or attempting automated access to the website is to use CAPTCHA to prove whether it is a real human who is accessing the website.


This project is intended to test the idea whether the bot can beat human in cracking simple CAPTCHA.


I am using fashion mnist dataset.


The dataset contains 28 by 28 grayscale images in 10 categories. There are 60,000 images in train set and 10,000 images in test set.


The 10 categories are as follows:

  1. T-shirt/Top
  2. Trousers
  3. Pullover
  4. Dress
  5. Coat
  6. Sandal
  7. Shirt
  8. Sneaker
  9. Bag
  10. Ankle boot


In this web application, I will be testing using random 30 images taken from 100 images. I have taken random 100 images out of the test set.


The model is trained using CNN (Convolutional Neural Network). I used all training examples to train the model and tested the model on 9900 images (excluded 100 random images). The model has a train accuracy of 0.9190 and test accuracy of 0.9111.
